HTML

● DOCTYPE:

Se utiliza para especificar la version de HTML que se esta utilizandoen la pagina

● html:

Define el inicio y el final de la pagina

● head:

Define la seccion de encabezado de la pagina, donde se incluyen elementos como el titulo de la pagina, metaetiquetas, scrips, entre otros

● title:

Define el titulo de la pagina web que aparece en la pestaña del navegador

● body:

Define la seccion del cuerpo de la pagina web, donde se incluyen todos los elementos que se mostraran en la pagina

● h1 a h6:

Define los encabezados o titulos de diferentes niveles de jerarquia en la pagina web. El uso de esta etiqueta formateará cualquier texto entre la etiqueta de apertura y la etiqueta de cierre como un Título o subtítulo. Por ejemplo, en Etiqueta de encabezamiento serían las etiquetas HTML y "Etiqueta de encabezamiento" es el elemento HTML, es decir, el encabezamiento de la página.

● a:

Define un enlace que el usuario puede hacer clic para ir a otra página web o a una sección diferente de la misma página.

● img:

Define una imagen que se mostrará en la página web.

● p "Etiqueta de parrafo /p:

Define un párrafo de texto. Las etiquetas "p" y "/p" son etiquetas HTML y la "Etiqueta de párrafo" es el elemento HTML, es decir, el texto de la página. Esta etiqueta formatea cualquier texto entre la etiqueta "p" de apertura y la etiqueta "/p" de cierre como un párrafo estándar o texto de cuerpo principal.

● b "Etiueta en negrita"/b:

Esta etiqueta dará formato de negrita a cualquier texto que se encuentre entre la etiqueta "b" de apertura y la etiqueta "/b" de cierre. Aquí, "b" y "/b" son las etiquetas HTML y la "etiqueta de negrita" es el elemento HTML, es decir, el texto de la página.

● i "Etiqueta en cursiva" /i:

Esta etiqueta formateará como cursiva cualquier texto entre la etiqueta de apertura "i" y la etiqueta de cierre "/i". Aquí, "i" y "i" son las etiquetas HTML y "Etiqueta cursiva" es el elemento HTML (el texto de la página).

● u "Etiqueta de subrayado" /u:

Esta etiqueta formateará cualquier texto entre la etiqueta "u" de apertura y la etiqueta "/u" de cierre como subrayado. Aquí, "u" y "/u" son las etiquetas HTML y "Etiqueta de subrayado" es el elemento HTML, es decir, el texto de la página.

● div:

Define una división o sección en un documento. Se utiliza para agrupar y estilizar contenido.

● span:

Define una sección en línea en un documento. Se utiliza para aplicar estilos a partes específicas del texto dentro de un párrafo u otro elemento de bloque.

● ul:

Define una lista desordenada. Se utiliza para crear listas de elementos donde el orden no es importante.

● ol:

Define una lista ordenada. Se utiliza para crear listas de elementos ordenados numéricamente o alfabéticamente.

● li:

Define un elemento de lista dentro de "ul" o "ol". Se utiliza para agregar elementos a una lista.

● table:

Define una tabla. Se utiliza para mostrar datos en filas y columnas.

● tr:

Define una fila en una tabla. Se utiliza para agrupar celdas de datos en una fila.

● td:

Define una celda de datos en una tabla. Se utiliza para agregar datos dentro de una fila y columna en una tabla.

● th:

Define una celda de encabezado en una tabla. Se utiliza para agregar encabezados a las filas o columnas de una tabla.

● form:

Define un formulario HTML. Se utiliza para recopilar datos del usuario, como nombres, contraseñas, direcciones, etc.

● input:

Define un campo de entrada en un formulario. Se utiliza para crear campos de texto, botones, casillas de verificación, etc.

● button:

Define un botón clickeable. Se utiliza para crear botones interactivos en formularios o para realizar acciones específicas en la página.

● texterea:

Define un área de texto multilínea. Se utiliza para permitir a los usuarios ingresar texto largo en un formulario.

● select:

Define una lista desplegable. Se utiliza para crear menús desplegables donde los usuarios pueden seleccionar una opción de una lista.

● option:

Define una opción dentro de un "select". Se utiliza para especificar las opciones disponibles en un menú desplegable.
